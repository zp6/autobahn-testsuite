# Jules OSS Bounty Program

## About
This repository participates in the Jules OSS bounty program, rewarding contributors for improvements.

## Eligible Contributions

### Bug Fixes ($25-200)
- Fix failing tests
- Resolve open bug reports
- Performance improvements

### Feature Implementation ($50-500)
- Implement requested features from issues
- Add new test protocols
- Improve documentation

### Security Fixes ($100-1000)
- Fix vulnerabilities
- Improve test coverage for security-critical paths
- Add fuzzing harnesses

## How to Participate
1. Find an issue labeled `jules-bounty`
2. Comment "I'm working on this"
3. Fork and create a feature branch
4. Implement the fix
5. Add tests
6. Submit PR with `Wallet: <address>` in description

## Review Process
1. Automated tests run
2. Code review within 7 days
3. Feedback provided if changes needed
4. Merge and payment within 14 days

## Payment
- Payments via GitHub Sponsors or crypto
- Include wallet in PR description
- Min payout: $25
